Network Administrator

  • Location: John St, Hawthorn VIC 3122
  • Job-Type: Full-Time
  • Department: It Administration
  • Reports To:IT Manager / Senior Network Engineer

Job Summary

The Network Administrator is responsible for installing, configuring, maintaining, and troubleshooting
network infrastructure to ensure optimal performance and security. This role involves managing LANs,
WANs, VPNs, firewalls, and cloud-based networks while providing technical support to internal teams
and clients.

Key Responsibilities

Network Infrastructure Management

  • Configure and Maintain:
    • Routers and switches
    • Firewalls and wireless access points
  • Manage LAN/WAN environments with VLANs, routing, and VPNs
  • Monitor network performance and resolve connectivity issues
  • Implement redundancy and failover solutions for uptime

Security and Compliance

  • Enforce network security using firewalls, VPNs, and IDS/IPS
  • Conduct patching, audits, and respond to incidents
  • Ensure compliance with relevant security standards (e.g., ISO 27001, NIST)

Cloud & Virtual Networks

  • Support networking in cloud environments (AWS, Azure)
  • Configure virtual networks in VMware/Hyper-V
  • Assist with SD-WAN and hybrid cloud setups

Support & Documentation

  • Provide Level 2/3 technical support
  • Document network topologies, configurations, and incidents
  • Generate network performance and security reports

Qualifications & Skills

Category Requirements
Education Degree in IT/Computer Science or equivalent experience
Experience 3+ years in network administration (MSP experience is a plus)
Core Skills VLANs, DHCP, DNS, VPN, firewalls, BGP/OSPF
Tools Cisco, Fortinet, Wireshark, PRTG, cloud platforms
Certifications Preferred: CCNA, Network+, Fortinet NSE
Soft Skill Strong troubleshooting, communication, documentation
